# Excel VBA CSV集計ツール

## 概要

CSVファイルを読み込み、データ検証・ログ出力・売上集計を行うExcel VBAツールです。

## 開発状況

🚧 開発中

### 実装済み機能

- CSVファイル読込
- UTF-8文字コードチェック
- CSVヘッダー検証
- データ型チェック
- 重複データ判定
- エラーログ作成
- 処理ログ作成

## 構成

### 標準モジュール

- ReadMain.bas
- TotalMain.bas
- Log.bas

### クラスモジュール

- Setting.cls
- Csv.cls
- Fso.cls
- Log.cls

## 設計方針

- 処理を責務ごとにモジュールへ分割
- 設定情報をクラスで管理
- CSV読込処理を共通化
- エラー処理を一元管理
- 処理ログ・エラーログを出力
- Enumによる列番号管理
- Dictionaryを利用した高速な重複チェック
- 保守性・再利用性を意識した設計
